## üöÄ Building for Production

### Build PWA
```bash
npm run build:pwa
```

This will:
1. Build optimized production version
2. Generate service worker
3. Create distribution package
4. Output to `dist/` folder

### Build for Desktop (Tauri)
```bash
npm run build:desktop
```

Creates native installers:
- macOS: `.dmg` file
- Windows: `.exe` file
- Linux: `.AppImage` or `.deb`

### Build for Android
```bash
npm run build:android
```

Creates `.apk` file for Android installation.

## üì¶ Distribution Methods

### Method 1: Web Hosting (PWA)
1. Build: `npm run build:pwa`
2. Upload `dist/` folder to web server
3. Ensure HTTPS is enabled (required for PWA)
4. Users visit URL and install via browser

### Method 2: Native Installers
1. Build: `npm run build:desktop` or `npm run build:android`
2. Distribute installer files (`.dmg`, `.exe`, `.apk`)
3. Users double-click to install

### Method 3: Package Distribution
1. Build: `npm run build:pwa`
2. Run: `npm run package`
3. Distribute `Grounded-PWA.zip` file
4. Users extract and serve locally or upload to hosting
