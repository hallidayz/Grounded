## Core Features

### 1. Values Selection & Ranking

**Location**: Values Tab

**Purpose**: Define your core values and establish your "North Star"

**How to Use**:
1. Tap the **Values** tab
2. Browse available values (up to 10 can be selected)
3. Tap values to select/deselect
4. **Drag to rank**: Your top value becomes your "North Star"
5. Values are saved automatically

**Tips**:
- Choose values that truly resonate with you
- Your top value guides Dashboard recommendations
- You can reorder values anytime by dragging

### 2. Dashboard - Daily Reflection

**Location**: Dashboard Tab

**Purpose**: Daily reflection on your values and progress

**How to Use**:
1. Open the **Dashboard** tab
2. See your values displayed as cards
3. **Tap a value card** to open detailed view
4. In the detail view:
   - **Log a Win**: Record something you did well
   - **Set Next Aim**: Create a micro-goal for tomorrow
   - **View History**: See past reflections
5. Use the AI coach for guidance (if models are loaded)

**Daily Practice**:
- Check Dashboard daily
- Log at least one win per day
- Set one small "Next Aim" per day
- Build consistency over time

### 3. Reports - Clinical Summaries

**Location**: Reports Tab

**Purpose**: Generate summaries for therapy sessions

**How to Use**:
1. Open the **Reports** tab
2. Select date range
3. Choose format:
   - **SOAP**: Subjective, Objective, Assessment, Plan
   - **DAP**: Data, Assessment, Plan
4. Tap **"Generate Report"**
5. Review and share with your therapist

**Therapy Integration**:
- Generate reports before sessions
- Share with your therapist for context
- Track progress over time
- Identify patterns and themes

### 4. Settings & Configuration

**Location**: Settings Tab (‚öôÔ∏è icon)

**Features**:
- **Reminders**: Enable notifications for daily check-ins
- **LCSW Configuration**: For therapists to customize treatment protocols
- **Theme**: Switch between light and dark mode
- **Data Management**: Export or clear your data

## Daily Workflow

### Recommended Daily Routine

**Morning (5 minutes)**:
1. Open Grounded
2. Check Dashboard
3. Review your North Star value
4. Set one "Next Aim" for today

**Evening (5-10 minutes)**:
1. Open Dashboard
2. Log at least one win from today
3. Reflect on how you embodied your values
4. Update "Next Aim" for tomorrow if needed

**Weekly (15 minutes)**:
1. Review your Reports
2. Generate a summary for therapy session
3. Reflect on patterns and progress
4. Adjust values if needed

## AI Features

### On-Device AI Processing

**Privacy**: All AI processing happens on your device. No data leaves your device.

**Model Loading**:
- Models download automatically on first launch
- Progress bar shows download status
- Models are cached for instant subsequent launches
- Works offline once downloaded

**AI Capabilities**:
- **Mood Tracking**: Assess emotional states
- **Counseling Coach**: Provide structured guidance
- **Reflection Prompts**: Suggest questions for deeper reflection
- **Goal Setting**: Help create meaningful micro-goals

**Fallback Mode**:
- If models fail to load, app uses rule-based responses
- Core functionality remains available
- AI features will be unavailable in fallback mode

### Using AI Features

**In Dashboard**:
- AI coach provides guidance when you open a value
- Suggestions for reflection questions
- Help with goal setting

**In Reports**:
- AI helps generate clinical summaries
- Identifies patterns and themes
- Suggests areas for discussion

## Therapy Integration

### For Clients

**Before Sessions**:
1. Generate a Report for the past week/month
2. Review your Dashboard entries
3. Note any patterns or concerns
4. Share Report with therapist if desired

**During Sessions**:
- Reference your values and goals

- Set new goals based on session insights

**After Sessions**:

## Settings & Customization

### Reminders

**Enable Notifications**:
1. Go to Settings
2. Toggle "Reminders" ON
3. Set frequency:
   - **Hourly**: Reminders every hour
   - **Daily**: Once per day at specified time
   - **Weekly**: Once per week
   - **Monthly**: Once per month
4. Set notification time (for daily/weekly/monthly)

**Notification Content**:
- Reminds you to check Dashboard
- Shows your North Star value
- Encourages daily practice

### Theme

**Switch Themes**:
- Tap the theme toggle in header
- Choose light or dark mode
- Preference is saved automatically

### Data Management

**Export Data**:
- Settings ‚Üí Export
- Download your data as JSON
- Keep backups for your records

**Clear Data**:
- Settings ‚Üí Clear Data
- Removes all local data
- Cannot be undone

## Troubleshooting

### AI Models Not Loading

**Symptoms**:
- Progress bar stuck
- AI features unavailable
- App shows "rule-based responses" message

**Solutions**:
1. Check internet connection (needed for first download)
2. Wait for download to complete (may take several minutes)
3. Restart the app
4. Check browser compatibility (see Help ‚Üí Debug Log)
5. App will work with rule-based responses if models fail

### App Won't Open

**Solutions**:
1. Restart your device
2. Clear browser cache (if PWA)
3. Reinstall the app
4. Check for updates

### Data Not Saving

**Solutions**:
1. Check browser storage permissions
2. Ensure you're logged in
3. Check available storage space
4. Try exporting data as backup

### Reports Not Generating

**Solutions**:
1. Ensure you have logged entries
2. Check date range selection
3. Try a different date range
4. Generate Debug Log for diagnostics

### Getting Help

2. **Debug Log**: Help ‚Üí Generate Debug Log
