### Build Desktop Installers (DMG/EXE)

```bash

```

**Output:**
- macOS: `src-tauri/target/release/bundle/macos/Grounded.dmg`
- Windows: `src-tauri/target/release/bundle/msi/Grounded_0.0.0_x64_en-US.msi`

### Build Android APK

**Prerequisites:** Android Studio and Android SDK must be installed

```bash
npm run build:android
```

**Output:**
- `android/app/build/outputs/apk/release/app-release.apk`

### Build Everything

```bash
npm run build:all
npm run package
```

This creates a complete distribution package with all installers.

## üìã Next Steps

### 1. Create Proper Icons (Required for Production)

Tauri needs `.icns` (Mac) and `.ico` (Windows) icon files. Currently using PNG placeholders.

**Option A: Use Online Tool**
1. Go to https://cloudconvert.com/png-to-icns (for Mac)
2. Go to https://cloudconvert.com/png-to-ico (for Windows)
3. Upload `public/pwa-512x512.png`
4. Download and place in `src-tauri/icons/`:
   - `icon.icns` (for Mac)
   - `icon.ico` (for Windows)

**Option B: Use Command Line (Mac)**
```bash
# Create .icns from PNG
iconutil -c icns src-tauri/icons/icon.icns --convert src-tauri/icons/128x128.png
```

### 2. Test Desktop Build

```bash
npm run build:desktop
```

First build will take 10-20 minutes (downloads Rust dependencies).
Subsequent builds are faster (2-5 minutes).

### 3. Test Android Build (If Android Studio Installed)

```bash
npm run build:android
```

**Note:** Requires Android Studio and Android SDK setup.

## üì± Distribution

### For End Users

**Easiest Method:**
1. Build installers: `npm run build:all`
2. Package everything: `npm run package`
3. Email the `Grounded-PWA.zip` file
4. Users extract and find installers in `installers/` folder
5. Users double-click to install - **that's it!**

### Installer Locations

After building:
- **DMG (Mac)**: `src-tauri/target/release/bundle/macos/Grounded.dmg`
- **EXE (Windows)**: `src-tauri/target/release/bundle/msi/Grounded_0.0.0_x64_en-US.msi`
- **APK (Android)**: `android/app/build/outputs/apk/release/app-release.apk`
